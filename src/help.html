Contextura Help

Hotkeys

  • Cmd + Shift + T - Toggle overlay visibility
  • Cmd + Shift + M - Manually clear context memory
  • Cmd + Shift + G - Switch to the next installed local model
  • Cmd + Shift + R - Force fresh translation
  • Cmd + Shift + Q - Quit

FAQ

Why isn't it appearing?
Check Screen Recording permission. Make sure you fully stop scrolling for at least 0.3 seconds.

Why does the overlay say capture is restarting?
Contextura now restarts the ScreenCaptureKit stream if frames stall after display sleep, wake, or a capture reset.

Why can model switching fail?
Cmd + Shift + G only works when at least two GGUF models are installed in the models folder.
